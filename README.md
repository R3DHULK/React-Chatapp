# React-Chatapp
Real Time Chatapp Using React, Node Js And Socket.io

Don't Use Whatsapp, make your own

## Pre-requisites :
Make Sure Nodejs is installed

for installing modules type :
```
npm install
```
## Git Installation

```
git clone https://github.com/R3DHULK/react-chatapp

cd react-chatapp

node server.js
```
